## 📂 Practical 02

### Programs Included:
1. Practical2.java  
2. q2.java  
3. q3.java  

### Description:
This practical implements a simple Student Evaluation Module in Java to help teachers analyze student performance.
It includes:
Using if-else statements to find the highest mark among three subjects.
Using the ternary operator to find the biggest number in a shorter way.
Calculating the average marks and assigning grades using a switch-case statement based on predefined grade ranges (O, A, B, C, D, F).
The practical demonstrates the use of conditional statements, decision-making, and basic logical operations in Java.

### Output:
Output screenshots are available in the `Practical_02` folder.

### How to Execute (Using NetBeans IDE):

1. Open NetBeans IDE  
2. Create or open a Java Application project  
3. Add a new Java class  
4. Paste the program code  
5. Click **Run (▶)**  
6. Check the Output Window for results
